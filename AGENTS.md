This file contains information for AI coding agents working on this project.

## Project Overview

Pi is a minimal terminal coding harness. This is a dummy/empty project for configuring and extending Pi. 

## Agent Instructions
- Consult the relevant Pi documentation before making changes that edit, configure, extend, or otherwise affect the Pi harness or its agent-facing resources.
- Use the subagent role that matches the task: discovery/context gathering via `scout` or `context-builder`, planning via `planner`, implementation via `worker`, review via `reviewer`, research via `researcher`, and decision/drift checks via `oracle`.
